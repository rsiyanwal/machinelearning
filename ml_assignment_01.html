Explain the Difference: SLR vs MLR vs Logistic Regression vs Polynomial Regression

This section is compulsory. You must explain the differences in words, not only in code.

Model Target type Shape learned Typical output Dataset used here
Simple Linear Regression Continuous number Straight line A predicted numerical value Salary_Data.csv
Multiple Linear Regression Continuous number Linear combination of multiple features A predicted numerical value 50_Startups.csv
Logistic Regression Class label Decision boundary for classes Probability, then class prediction Social_Network_Ads.csv
Polynomial Regression Continuous number Curve A predicted numerical value Position_Salaries.csv
Task 7.1 In your own words, answer all five:
  1. Why can’t we use simple linear regression for the purchase prediction task?
  2. How is multiple linear regression different from simple linear regression?
  3. Why does polynomial regression perform better than a straight line on the salary-by-level dataset?
  4. Why are both SLR and polynomial regression still called regression?
  5. Why is logistic regression called “regression” even though it is used for classification?
A useful mental model: SLR asks, “What straight-line trend predicts a number from one feature?” MLR asks, “How do several features combine to predict one number?” Logistic regression asks, “What boundary separates two classes?” Polynomial regression asks, “Can I bend the input space so a linear model can capture a curve?”
